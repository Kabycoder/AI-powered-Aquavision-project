# Aquavision

AI-powered Aquavision — a project for monitoring and visualizing water quality.

## Overview

This repository contains the Aquavision project source. The `backend` directory contains the server-side code (Python).

## Quickstart (Windows)

Prerequisites:
- Python 3.8+
- `git`

Commands:

```powershell
python -m venv venv
venv\Scripts\Activate.ps1
pip install -r requirements.txt   # if present
python backend\main.py
```

## Files
- `backend/` — server code
